Computes the location of the player cell targeted by a missile The computation is done at medium level\-: this method searches for a zone in the player board, in which the ship with the greatest size ($>$1) that has not been already destroyed, could lie. 

Note\-: this method could be optimized since every time a missile is going to be launched, the whole copy board is scanned for collecting every cell representing an interesting zone. Since only a relatively small part of the board is modified after each missile launch, it might be better to modify incrementally the collection of zones, instead of recreating a collection each time. But the code would be complexified, we prefer to use the \char`\"{}brute-\/force\char`\"{} solution.
