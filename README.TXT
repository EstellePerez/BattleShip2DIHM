Contents

The BattleShip2D project contains:
- this file
- the battleship2D directory containing the source code (except for the UI part)
- the jar archive name BattleShip2D.jar
- the META-INF directory used to create the BattleShip2D.jar
- the COPYING file provided to comply to GPL3 license
- the Documentation directory, which provides every additional information

To compile and run BattleShip2D, please follow the Documentation/manual.pdf file recommendations
